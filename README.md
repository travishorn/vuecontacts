# vuecontacts

> Showcasing Vue.js, vue-router, and vuex through a simple CRUD application.

## Installation

    > git clone https://github.com/travishorn/vuecontacts
    > cd vuecontacts
    > npm install

## Run (development)

Start the JSON API server, then serve UI with hot reload at localhost:8080

    > npm run server
    > npm run dev

## Build

    > npm run build
